Шаг 1

Обязательные параметры

Эти значения необходимы для авторизации и работы граббера. Без них система не запустится.

1

api_id и api_hash

Регистрируйтесь на my.telegram.org и получите API credentials в разделе "API development". api_id — это число, api_hash — буквенно-цифровая строка.

Обязательно Видео-инструкция
2

bot_token

Получите токен бота через @BotFather в Telegram. Отправьте команду /newbot, придумайте название и username, сохраните токен. Без него не будут работать команды управления.

Обязательно Видео-инструкция
3

my_id и technical_channel_id

Узнайте свой ID через @userinfobot. Создайте технический канал для логов и узнайте его ID (начинается с -100). Эти параметры нужны для уведомлений и мониторинга.

Обязательно Видео-инструкция
Шаг 2

Необязательные параметры

Расширенные настройки для интеграции с нейросетями и тонкой настройки поведения граббера.

1

timezone_offset

Смещение часового пояса в минутах относительно UTC. Используется для корректного планирования автопостинга. Например, для Москвы (UTC+3) укажите 180.

2

openai_api_key_gpt

API ключ OpenAI для использования GPT моделей. Получите на platform.openai.com. Необходим для рерайта текстов и генерации контента через GPT-4.

3

openai_model_gpt

Название модели GPT для генерации текста. По умолчанию "gpt-4o-mini". Доступны: gpt-4o, gpt-4o-mini, gpt-4-turbo.

4

external_model_gpt

URL для интеграции со сторонними GPT API (например, One API, LiteLLM). Позволяет использовать альтернативные провайдеры с совместимым API.

5

openai_api_key_dalle

API ключ OpenAI для генерации изображений через DALL·E. Может быть тем же ключом, что и для GPT, или отдельным.

6

openai_model_dalle

Модель для генерации изображений. По умолчанию "dall-e-3". Доступна также "dall-e-2" (более дешевая, но менее качественная).

7

mistral_api_key

API ключ Mistral AI для генерации текстов через их модели. Получите на console.mistral.ai.

8

external_model_mistral / chat_mistral_model

Настройки для внешних моделей Mistral. Укажите URL API и название модели (например, "mistral-medium", "mistral-small").

9

flux_api_key

API ключ для Flux — альтернативного сервиса генерации изображений. Используется вместо или в дополнение к DALL·E.

10

giga_api_key

API ключ GigaChat от Сбера для генерации текстов и изображений. Получите на developers.sber.ru.

11

Системные параметры

Дополнительные настройки для работы с изображениями и RSS: 

{
  "FAVIC_HEADER_LIGHT": 220,
  "FAVIC_HEADER_DARK": 133,
  "TRIGGERS": {
    "rss": []
  }
}

FAVIC_HEADER_LIGHT/DARK — пороги яркости для определения светлого/тёмного фона фавикона.
TRIGGERS.rss — массив RSS-источников для мониторинга.
