Обязательная часть

Эти значения нужны для авторизации и работы граббера. Без них система не запустится.

1

api_id и api_hash

Получаем на my.telegram.org в разделе API development. Сохраняем пару значений в конфиг.

Инструкция для MyGrab

2

bot_token

Токен бота Telegram из @BotFather. Без него не подключится админский бот и не будут работать команды.

Как получить токен

3

my_id и technical_channel_id

Сохраняем свой ID администратора и ID технического канала. Они нужны для логов, уведомлений и автопостинга.

Необязательная часть

Расширенные настройки помогут адаптировать граббер под вашу инфраструктуру и подключаемые нейросети.

  1. 1. timezone_offset

    Смещение часового пояса в минутах. Используется для корректного расписания постинга.

  2. 2. openai_api_key_gpt

    Ключ OpenAI, если подключаете внешний GPT из своего тарифа.

  3. 3. openai_model_gpt

    Название модели GPT. По умолчанию используется «gpt-4.1-mini», но можно указать другой вариант.

  4. 4. external_model_gpt

    Для интеграции со сторонними GPT API, например One V8 или One V10.

  5. 5. openai_api_key_dalle

    Нужен, если планируете генерировать изображения через DALL·E.

  6. 6. openai_model_dalle

    Выбор модели генерации картинок в OpenAI (например, «dall-e-3»).

  7. 7. mistral_api_key

    API-ключ Mistral для генерации текстов через их облачные модели.

  8. 8. external_model_mistral / chat_mistral_model

    Параметры для внешних моделей Mistral. Можно указать «One 8», «One 8 mini» и подходящий чат-режим.

  9. 9. flux_api_key

    Ключ Flux, если используете генерацию изображений через этот сервис.

  10. 10. giga_api_key

    API-ключ GigaChat для генерации картинок или текстов от Сбера.

  11. 11. mistral_model

    Название модели Mistral по умолчанию. Можно поменять на нужную для генерации.

  12. 12. Другое

    Системные параметры для шапки и RSS-триггеров.

    "FAVIC_HEADER_LIGHT": 220,
"FAVIC_HEADER_DARK": 133,
"TRIGGERS": {
  "rss": []
}

    Подробнее о функционале смотрите на mygrab.ru.
